# Higher or Lower Fútbol

## Integrantes

- Lorenzo Beccaria
- Franco Viggiano
- Lucio Rosenthal
- Thiago Robles

## Descripción

Este proyecto consiste en un juego web basado en el valor de mercado de jugadores de fútbol.

El usuario deberá adivinar si un jugador tiene un valor de mercado mayor o menor que otro.

El sistema contará con:

- Login
- Registro
- Juego Higher or Lower
- Ranking
- Panel de administrador

## Tecnologías

- HTML
- CSS
- JavaScript
- Node.js
- MySQL
- GitHub
